üìÑ Project Overview

This project focuses on the quantitative assessment of a company's financial health through Ratio Analysis. Using an integrated 3-statement model as the foundation, this analysis evaluates liquidity, solvency, operational efficiency, and profitability over a 7-year horizon (3 years of historical data and a 5-year forecast). The goal is to transform standard accounting figures into meaningful insights regarding the company's risk profile and growth potential.
